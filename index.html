Historia de HTML y CSS

Versión Año Descripción
HTML 1.0 1991 Tim Berners-Lee inventa HTML para crear y compartir documentos en la World Wide Web. Esta primera versión permitía el formato básico de texto, enlaces y listas.
HTML 2.0 1995 En 1995, la IETF publica la especificación HTML 2.0. Esta versión introdujo formularios y soporte para scripts, lo que marcó un avance significativo en la funcionalidad de las páginas web.
HTML 4.01 1999 1999 - HTML 4.01 fue una versión crucial que introdujo mejoras significativas en la web. Además de la introducción de formularios, esta versión también brindó un sólido soporte para hojas de estilo CSS, permitiendo un diseño más avanzado y la separación efectiva de la presentación y el contenido en las páginas web.
XHTML 1.0 2000 XHTML 1.0 representó un paso hacia la estandarización y la rigurosidad sintáctica. Esta versión de HTML adoptó la sintaxis XML, lo que facilitó su integración con otras tecnologías basadas en XML. XHTML 1.0 fomentó buenas prácticas de codificación y marcó un hito en la evolución del lenguaje web.
HTML5 2014 HTML5 marcó una revolución en el desarrollo web al proporcionar un conjunto completo de nuevas características y API. Esta versión mejoró el soporte multimedia incorporando elementos de audio y video nativos, introdujo APIs para interactividad avanzada y estructuras semánticas que facilitan la comprensión del contenido para los motores de búsqueda.
CSS 1 1996 CSS 1 fue la primera versión de CSS y desempeñó un papel crucial al permitir la separación de la presentación y el contenido en el desarrollo web. Introdujo propiedades básicas de estilo, como colores y márgenes, allanando el camino para la creación de diseños más atractivos y flexibles.
CSS 2.1 2011 Representó una consolidación y refinamiento de las capacidades de estilo de CSS 2. Se centró en corregir errores y mejorar la interoperabilidad entre navegadores, brindando a los desarrolladores una plataforma más sólida para crear diseños web coherentes y compatibles.
CSS3 Actualidad CSS3 marcó una evolución continua en las capacidades de diseño web al introducir módulos independientes. Estos módulos permiten a los desarrolladores elegir y utilizar características específicas según sus necesidades, brindando una mayor flexibilidad y la capacidad de implementar diseños más avanzados y ricos en funcionalidades.

Cuadro Comparativo: Lenguaje de Programación vs HTML

Característica Lenguaje de Programación HTML
Sintaxis Utiliza una sintaxis estructurada con reglas específicas de programación. Utiliza una estructura de etiquetas y atributos para definir la estructura y contenido de una página web.
Propósito Diseñado para la creación de programas y la manipulación de datos. Diseñado para la creación de páginas web y la presentación de contenido en internet.
Ejecución Es interpretado o compilado para ejecutarse en un entorno de ejecución específico. Es interpretado por el navegador web para mostrar la estructura y contenido de una página web.
Variables Permite la declaración y manipulación de variables para almacenar datos. No tiene variables en el sentido tradicional; utiliza atributos y contenido para representar datos.
Funciones Permite definir y llamar funciones para realizar tareas específicas. Utiliza elementos y atributos específicos para definir la estructura y presentación de la página.
